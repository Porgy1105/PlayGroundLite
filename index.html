基本的な要素の取得

            
				document.getElementById('parent'); //id名で取得
				document.getElementsByClassName('parent'); // クラス名で取得
				document.getElementsByTagName('div'); // タグ名で取得
				document.querySelector('.parent'); // 最初にヒットした要素が取得される
				document.querySelectorAll('.parent'); // ヒットした要素全部取得

使い方

document.getElementsByClassName("parent")[0].innerText;

取得した要素から子要素を取得
